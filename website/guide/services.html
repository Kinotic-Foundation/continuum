Continuum Services ​

Continuum applications are built on Continuum services, which are created using Spring and deployed as Spring Boot applications. Unlike standard Spring services, Continuum services are automatically discoverable and can transparently communicate with each other. In this page, we will explore how Continuum services are discovered and interconnected for seamless communication.
